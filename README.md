# Chords - Demo Library (Public Domain)

This repository contains a demo database of **over 1700 public domain songs** (chords and lyrics) designed to be used with the open-source music app **[Chords](https://github.com/phi-k/chords)**.

For legal reasons and to respect copyright laws, the Chords application is distributed "empty" by default (much like a video player or a web browser). This library allows you to immediately test all of the app's features with 100% legal content.

## 📦 What's inside?

The `chords_demo_library.json` file (~4 MB) contains **1720 public domain works**, meticulously formatted for the Chords rendering engine.

You will find:
* 🎻 **Traditional Music** (American Folk, European ballads...)
* 🎄 **Christmas Carols** (Jingle Bells, Silent Night...)
* 🧸 **Children's Songs** (Alouette, Twinkle Twinkle...)
* ⛪ **Old Hymns & Gospel**

*Each song includes its metadata (Key, Capo, Tuning) as well as clean chord formatting above the lyrics.*

## 🚀 How to use it in Chords?

It's very simple! Importing takes just 3 clicks thanks to Chords' embedded local database (Isar):

1. **Download the demo file**  
   👉 **[Click here to download `chords_demo_library.json` (Direct Link)](https://raw.githubusercontent.com/phi-k/chords-demo-library/main/chords_demo_library.json)** *(Right-click > "Save link as..." if the file opens directly in your browser)*.

2. **Open the Chords app**  
   If you don't have it yet, download it from the [main Chords repository](https://github.com/phi-k/chords/releases).

3. **Import the library**  
   Go to `Settings` ⚙️ > `Backups` > `Import library`.  
   Select the JSON file you just downloaded.  

🎉 **That's it!** Your app now contains 1700 songs playable offline, with absolutely no ads. You can now test transposing, chord simplification, dark mode, the metronome, and PDF generation!
